---
title: React Native Boost
description: A Babel plugin and runtime toolkit for safer React Native micro-optimizations.
---

React Native Boost improves React Native render performance by replacing safe `Text` and `View` usages with their native counterparts during compilation.

It is designed to be safety-first: if the plugin cannot prove a transformation is safe, it skips it.

<Callout type="warn" title="Experimental">
  React Native Boost is still experimental. If you hit issues, open a reproducible report on GitHub and use the plugin's
  ignore/disable options for temporary workarounds.
</Callout>
